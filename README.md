# Khareej.net Project

## Setup Instructions

### Backend (Server)
1. Navigate to `server/` folder.
2. Run `npm install`.
3. Create `.env` file based on `.env.example`.
4. Run `npm run dev`.

### Frontend (Client)
1. Open `client/index.html` in browser or use Live Server.

### Database
1. Import `database/schema.sql` into MySQL.
